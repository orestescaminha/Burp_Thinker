Burp Thinker

API local para análise de requisições/respostas. Endpoints disponíveis:

  • POST /analyze/request
  • POST /analyze/response
  • POST /payloads/sqli
  • POST /jwt
  • GET /tasks/{task_id}

Documentação automática (Swagger): /docs — útil para testar interativamente.

Para uso com Burp, mantenha o servidor atrelado a 127.0.0.1 e configure o token em BURP_THINKER_TOKEN.
